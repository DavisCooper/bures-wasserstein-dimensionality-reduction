This repo compares SPD dimensionality reduction techniques using several different distance functions on SPD manifolds. 
The novel contribution is the dimensionality reduction using the fixed-rank spd manifold distance. 

This repo extends source code available from: 
- https://sites.google.com/site/mehrtashharandi/publications - Dimensionality Reduction on SPD Manifolds: The Emergence of Geometry-Aware Methods 
- https://github.com/andyjm3/GBW
